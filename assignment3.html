Assignment 3

Description

For this assignment I continued experimenting with composition and motion in p5.js. The sketch focuses on layering simple shapes and letting them move or repeat over time to create a calm, abstract animation. Instead of telling a direct story, it is more about watching how the screen slowly changes.

This project helped me think about timing, spacing, and how small variations in position, size, or color can make the whole image feel different. It is a study in building atmosphere using very limited elements.
